# Sales Performance Dashboard – Excel Project

## Project Overview
This is a practice data analytics project built in Microsoft Excel to demonstrate data cleaning, analysis, and dashboard creation skills.

## Objectives
- Clean and organize raw sales data
- Analyze sales performance
- Identify trends and patterns
- Build an interactive dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas

## Key Insights
- Identified top-performing categories
- Observed sales trends over time
- Built a dashboard for visualization
